¿Qué es Cloud Computing?

Cloud computing es el suministro de recursos informáticos a través de internet, como servidores, almacenamiento, bases de datos, software y análisis, a los que se accede bajo demanda y pagando solo por el uso.

¿Qué es SaaS, PaaS e IaaS?

SaaS, PaaS e IaaS son los tres modelos principales de servicios de computación en la nube. IaaS (Infraestructura como Servicio) ofrece recursos de infraestructura de TI como servidores y almacenamiento virtualizados. PaaS (Plataforma como Servicio) proporciona un entorno para desarrollar, ejecutar y gestionar aplicaciones sin gestionar la infraestructura subyacente.

¿Qué es Virtualización?

La virtualización es la creación de versiones simuladas de recursos físicos de computación, como servidores, almacenamiento, redes o sistemas operativos, usando software para ejecutar múltiples entornos virtuales en una única máquina física.

¿Qué es Docker?

Docker es una plataforma de código abierto que permite crear, desplegar y ejecutar aplicaciones en contenedores.
